# WorkoutPAL_beta
Version beta du projet WorkoutPAL https://github.com/r00tp/WorkoutPAL

## Plan de développement

| Sprint    | Contenu | Livrable |
| -------- | ------- | ------- |
| S1 | Shell PWA + navigation + design system | App installable, 4 écrans vides navigables |
| S1 | Data layer + bibliothèque d'exercices (15 exos) | Affichage de la liste avec SVGs |
| S3 | Générateur de séances + écran séance | Lancer une séance complète |
| S4 | Timer multi-mode + sons | Timer fonctionnel dans la séance |
| S5 | Stockage historique + tracking reps/poids/RPE | Enregistrer et consulter une séance |
| S6 | Moteur de progression + dashboard | Recommandations basées sur l'historique |
| S7 | Polish : animations, swipe, wake lock, offline | App finale |
